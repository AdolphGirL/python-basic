#### 相關參考知識
- if and 判斷式  
  ```
  a = 10
  b = 5
  c = 1

  if a==b and a==c:
    print('YES')
  else:
    print('NO')

  if a!=b and a!=c:
    print('都不相等')

  ```

- list串列  
  ```
  如果需要宣告5種變數
  a = 1
  b = 2
  c = 3
  d = 4
  e = 5
  
  改
  x = [1, 2, 3, 4, 5]

  添加
  x.append(6)

  讀取某個位置的值
  x[0]

  列印
  for item in x:
    print(item)
  ```
